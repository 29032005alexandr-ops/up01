# UI Style Notes (по «Руководству по стилю»)

## Шрифт
- Segoe UI

## Цветовая палитра
- Основной фон: #FFFFFF
- Доп. фон/панели: #F4E8D3
- Акцент (кнопки/выделение): #67BA80

## Графика
- Логотип на главной форме
- Иконка приложения (если платформа позволяет)

## Общие правила
- Единый стиль на всех формах
- Заголовки окон по назначению
- Контролы не должны «выползать» за границы; учитывать разные DPI
