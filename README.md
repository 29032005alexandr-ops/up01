# УП.01 (8 семестр) — проект

## Структура
- `db/` — скрипты БД, ER-диаграмма, бэкап
- `data/raw/` — исходные файлы заказчика (как получены)
- `data/app_import/` — подготовленные к импорту файлы (очищенные/нормализованные)
- `app/` — исходный код приложения
- `docs/` — блок-схема, Use Case, тестовые сценарии, руководство пользователя
- `build/` — исполняемые файлы/релизы (если требуются отдельно)
- `report/` — отчёт по практике (DOCX/PDF)
